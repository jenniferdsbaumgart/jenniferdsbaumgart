## 👩‍💻 About Me

Chemist-turned-engineer. I traded molecular structures for system architectures, but the obsession stayed the same: understanding how things really work, and making them work better for people.

I build AI-powered products end to end — from the RAG pipeline that retrieves the right knowledge, to the interface that makes it feel obvious. I care about both, because a brilliant backend wrapped in a confusing UI is still a failure.

My work tends to live in domains where software has real stakes: health tech, mental health, and financial tools for underserved users. If you're building something that genuinely matters to the people using it, we'll probably get along.

Senior Software Engineer · Applied AI · Human-centred product thinking

## 🚀 Featured Projects

### 🧠 Noesis – Strategic Knowledge Platform
A full-stack RAG-powered knowledge platform. Organisations connect their data sources — APIs, documents, databases — and get a unified semantic search layer with AI-generated answers, source citations, and an embeddable widget for any site.

Built with a multi-tenant SaaS architecture, resilient background sync, and role-based access control across all layers.

- **Tech:** NestJS · Next.js · TypeScript · pgvector · BullMQ · OpenAI · MinIO · Docker
- **Status:** Private repo · [Request access or demo](mailto:jenniferdsbaumgart@gmail.com)

### 🧵 Texlink – Textile Production Management Platform
A B2B SaaS platform connecting fashion brands with garment manufacturers (facções). Features order management, production capacity tracking, compliance & document management, supplier visibility rules, and a supplier marketplace — built for the Brazilian textile supply chain.

- **Tech:** React · TypeScript · Next.js · Tailwind CSS · NestJS · PostgreSQL · Prisma · Docker
- **Status:** Active development · Private repo


### 💰 Monicount – Finances for Variable Income
A calm, low-friction finance app designed around neurodivergent brains and unpredictable income patterns, with AI-powered insights.

- **Tech:** React · Next.js · TypeScript · Tailwind CSS · OpenAI API · PostgreSQL · Stripe
- **[Live Demo](https://monicount.vercel.app/)** · **[Repository](https://github.com/jenniferdsbaumgart/monicount-fullstack-finances-app)**


### 🧭 Compath – AI Business Discovery
